Analyse de l’impact de l’envoi de SMS de rappel sur le taux de présence aux rendez-vous médicaux, à partir de données de santé réelles (open data).

🔍 Problématique
Évaluer si un rappel par SMS permet de réduire significativement le taux de rendez-vous non honorés (“no-show”) et d’améliorer l’efficacité organisationnelle des structures de santé.

🧪 Approche analytique

Mise en place d’un test A/B :

Groupe contrôle : patients sans SMS

Groupe traitement : patients avec SMS

Analyse comparative des taux de présence entre les deux groupes

📈 Méthodes statistiques utilisées

Statistiques descriptives et analyse exploratoire (EDA)

Tests d’hypothèses (test du χ² / test de proportions)

Régression logistique pour modéliser la probabilité d’absence

Analyse par sous-groupes (âge, sexe, délai de rendez-vous)

📊 Résultats & enseignements

Mesure de l’effet statistique du SMS sur le taux de présence

Identification de facteurs influençant l’absentéisme

Interprétation rigoureuse des résultats statistiques pour appuyer la prise de décision
